# Flandre.Adapters.OneBot

基于 [OneBot](https://github.com/botuniverse/onebot) 协议实现的 QQ 协议适配器，主要对 [go-cqhttp](https://github.com/Mrs4s/go-cqhttp) 提供支持。同时基于 go-cqhttp 对 QQ 频道也进行了一定的支持。
